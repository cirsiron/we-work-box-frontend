## 工作平台
  - 致力打造一套适合每个人自己的工作流，享受在工作中行云流水的感觉

### 功能点：
  - 1. 基于谷歌浏览器扩展，简洁易用

  - 2. 支持一键搜索，省去点击查找的麻烦

  - 3. 集合常用工作分类，可自定义分类

  - 4. 支持todolist

  - 5. 支持日历，并与todo打通

  - 6. 支持热门功能，软件，项目等的推荐


### 技术栈
  - 前端:
    - vue + element + vue-router + vuex
  - 后端:
    - koa + koa-router
    - 数据库：elasticsearch
  - 部署:
    - docker
